[![wakatime](https://wakatime.com/badge/github/ishkabar/ogurbot-sentinel.svg?style=flat-square)](https://wakatime.com/badge/github/ishkabar/ogurbot-sentinel)

![DevExpress](https://img.shields.io/badge/DevExpress-23.2-FF7200?style=flat-square)

DevExpress WPF desktop client for real-time boss timer display with color-coded warnings.

See [Desktop Client User Guide](../../installer/README.md) for installation and usage instructions.

## Features
- Real-time timer sync (5-300s intervals)
- Color-coded warnings (green/orange/red)
- Time offset adjustment (-180 to +180s)
- Always-on-top window
- MSI installer with WiX Toolset

## Dependencies
- `DevExpress.Wpf.Core` (23.2.14)
- `DevExpress.Wpf.Themes.Office2019Colorful` (23.2.14)
- `DevExpress.Mvvm.CodeGenerators` (22.1)
